##### Requirments
python3  
pytesseract  
opencv-python  
##### Install

```  
  pip install pytesseract  
```  

```
   pip install opencv-python  
```  
##### Usage  
Recognize email by memberId  
```
   python3 scrap.py -m 155412  
```  
    
or recognize email from image  
   
```
   python3 scrap.py -f email_image.gif  
```
